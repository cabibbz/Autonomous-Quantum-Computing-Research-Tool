# Sprint 010 — GME Witnesses: Efficient Entanglement Detection

**Date:** 2026-03-31
**Goal:** Can we detect entanglement class (GHZ/W/Cluster) from a few Pauli measurements instead of full state tomography?

## Motivation

We've spent 9 sprints characterizing entanglement structure with full density matrix access: entropy, MI, I3, discord, concurrence, negativity, Phi, noise fingerprints. But in practice, full tomography scales as 3^n measurements. Entanglement witnesses detect entanglement from a *constant* number of measurement settings.

Key questions:
1. Can we construct witnesses that detect GME for each archetype?
2. How specific are they — does a GHZ witness fire on W or Cluster states?
3. Can we build a classifier from Pauli expectation values alone?
